🔥 O objetivo desta API é fornecer ao FLP Foods Frontend os serviços de autenticação, gerenciamento de mesas e reservas. Este projeto foi desenvolvido logo após a conclusão dos cursos que realizei na Rocketseat, com a intenção de colocar em prática todos os conhecimentos adquiridos até então.

Durante o desenvolvimento, tive a oportunidade de entender melhor como funciona a estrutura de comunicação entre os diferentes módulos da aplicação, como Controllers, Services e Middlewares. Além disso, trabalhei na integração e no gerenciamento dos dados utilizando um ORM para a comunicação com o banco de dados, o que contribuiu bastante para fortalecer minha base em desenvolvimento backend e boas práticas de arquitetura.
